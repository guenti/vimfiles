# MyVim

## Introduction
That's my personal Vim setup. Use this at your own risk. :)

## Installation
```bash
$ mkdir $HOME/.vim
```

```bash
$ git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
```

```bash
$ vim +PluginInstall +qall
```

Happy Vim-ing. :)
